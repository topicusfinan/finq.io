Finq.io

Behavior based testing

In the early stages of development, stay tuned

To finally get rid of the need to use complex tools for use-case testing in enterprise environments. Finq.io introduces behavior based testing straight from the browser. A highly performant and scalable tool that will actually make testing fun!

Finq.io is focussed on collaboration to ensure that a team enjoys working together to test the quality of the software they build. Behavior-driven development and domain specific language for test specification, allow tests to become easy to:

  • Write

    No need to keep separate documentation to explain your tests. You write them in the same language as your usecases.

  • Run

    Focus on a single scenario in debug mode, or run a whole collection of tests at once on a target environment.

  • Organize

    Tag them and make sets of scenarios that you want to execute on specific occasions or for specific builds.

  • Report

    Get immediate access to tests ran by other members in your team and keep track of a run history for scenarios.
