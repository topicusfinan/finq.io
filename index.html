Finq.io

Behavior-driven development

In the early stages of development, stay tuned

To finally get rid of the need to use complex tools for use-case testing in enterprise environments. Finq.io introduces behavior-driven testing straight from the browser. A highly performant and scalable tool that will actually make testing fun!

Inspired by JBehave, Finq.io is focussed on collaboration, usability and test quality, and by combining behavior-driven development (BDD), domain-specific language (DSL), and test-driven development (TDD), tests to become easy to:

  • Write

    No need to keep separate documentation to explain your tests. You write them in the same language as your user stories.

  • Run

    Focus on a single scenario in debug mode, or run a whole collection of tests at once on a target environment.

  • Organize

    Tag them and make sets of scenarios that you want to execute on specific occasions or for specific builds.

  • Report

    Get immediate access to tests ran by other members in your team and keep track of a run history for scenarios.
